== Description ==

Easily display Google Calendar Events on your Wordpress site.

== Shortcodes ==

[dpi_calendar]

arguments:
*   "format" - the rendered view of the calendar. Accepts "month", "week", "day", "agendaWeek", and "agendaDay". Defaults to "month".

[dpi_gcal_events]

arguments:
*   "max_events" - the maximum number of events to display. Accepts any non-negative whole number.
